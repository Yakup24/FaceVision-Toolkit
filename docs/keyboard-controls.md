# Keyboard Controls

Keyboard controls are handled by `facevision_toolkit.runtime.handle_key`.

| Key | Action |
| --- | --- |
| `q` | Quit |
| `ESC` | Quit |
| `s` | Save screenshot |
| `e` | Toggle eye detection |
| `m` | Toggle mirror mode |

Headless mode skips OpenCV window and keyboard handling.
